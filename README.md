# RAMMB/CIRA SLIDER CLI

A command-line implementation of the RAMMB/CIRA SLIDER tool in Golang.

This is not an official product of NOAA, RAMMB, or CIRA.

## Example Usage

```bash
slider-cli --satellite=goes-16 --sector=conus --product=band-1
```
